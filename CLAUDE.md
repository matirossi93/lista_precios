# App Listas de Precios

## Stack
- Vanilla JS + CSS (sin framework, sin TypeScript)
- Express (server.js)
- Deploy: Hostinger EasyPanel (Docker)
- Data: CSV (jujuy.csv, raw_data.csv)

## Comandos
- `npm run dev` — dev local
- `npm start` — producción

## Reglas
- Stack mínimo, no agregar complejidad innecesaria
- Tiene Dockerfile + nixpacks.toml para deploy
